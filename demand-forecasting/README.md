# 📈 Demand Forecasting using Time Series

## Objective
To forecast future demand trends using historical time-series data.

## Approach
- Loaded and cleaned time-series dataset
- Converted data into Prophet-compatible format (ds, y)
- Built forecasting model using Facebook Prophet
- Generated future predictions for the next 30 days

## Results
- Successfully captured seasonal patterns and trends
- Generated interpretable future forecasts
