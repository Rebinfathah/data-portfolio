# 🚨 Anomaly Detection using Isolation Forest

## Objective
To identify unusual patterns and outliers in operational data.

## Approach
- Loaded dataset containing normal and extreme values
- Applied Isolation Forest algorithm for anomaly detection
- Classified data points as normal or anomalous
- Visualized anomalies using scatter plot

## Results
- Successfully detected extreme values as anomalies
- Clearly distinguished outliers from normal data points
