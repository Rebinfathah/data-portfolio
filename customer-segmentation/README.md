# 👥 Customer Segmentation using K-Means Clustering

## Objective
To segment customers based on income and spending behavior for targeted decision-making.

## Approach
- Loaded and cleaned customer dataset
- Selected key features: Income and Spending Score
- Applied K-Means clustering algorithm
- Visualized customer segments using scatter plot

## Results
- Identified 5 distinct customer groups
- Clear separation of clusters based on behavior patterns
